# kevinyan.com

My personal website built with React, TypeScript, and Tailwind CSS.

## Stack

- **React + Vite + TypeScript**
- **Tailwind CSS** — styling
- **Framer Motion** — animations
- **Cloudflare Pages** — hosting

## Sections

- About — bio and profile
- Career — work history, skills, certifications
- Personal — hobbies and photos
- Contact — mailto-based contact form

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Deployed automatically via Cloudflare Pages on push to `main`.
